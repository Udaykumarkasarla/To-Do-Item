# React To-Do List App

This is a simple to-do list application built using React and Vite.

## Features
- Add tasks
- Mark tasks as complete
- Edit tasks
- Delete tasks

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. Open browser at `http://localhost:5173`
